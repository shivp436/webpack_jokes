Packages Used

  1. Webpack
      Webpack is a module bundler
      it is used to create the bundle
      modules: webpack, webpack-cli
  2. HtmlWebpackPlugin
      HtmlWebpackPlugin simplifies creation of HTML files to serve your webpack bundles
      modules: html-webpack-plugin
  3. SCSS and CSS Loaders
      style-loader, css-loader, sass-loader
      sass-loader: helps in compiling the scss into css
      css-loader: Takes the CSS files generated by sass-loader (or directly provided CSS files) and translates them into a format that Webpack can understand.
      style-loader: Injects the final CSS into the DOM by adding a <style> tag.
